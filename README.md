# 🌍 Interslavic Translation for Minecraft Beta 1.7.3

[🌐 English](README.md) | [🌐 Medžuslovjansky](PROČITAJMENE.md) | [🌐 Меджусловјанскы](ПРОЧИТАЈМЕНЕ.md)

Welcome to the repository for the **Minecraft Beta 1.7.3** fixed Interslavic translation. This project aims to bring the classic Minecraft experience to speakers and learners of the Interslavic language (*Medžuslovjansky* / *Меджусловјанскы*).

---

## 🗺️ Supported Scripts

To ensure maximum accessibility across the Slavic community, this localization supports both official writing systems:
* **Latin Alphabet (Medžuslovjansky)**
* **Cyrillic Alphabet (Меджусловјанскы)** 

---

## ⚙️ Installation Guide

### For Players
1. **Install Mod:** First, you need to install the [Translated Legacy](https://modrinth.com) mod for Babric.
2. **Locate Folder:** Navigate to your configuration directory: `.minecraft/config/translated-legacy/b1.7.3/`
3. **Add Language File:** Choose your preferred script from this repository and copy it into that folder:
   * **For Cyrillic:** Drop `isv-c.json` there and rename it to `isv.json`
   * **For Latin:** Drop `isv-l.json` there and rename it to `isv.json`
4. **Activation:** Launch the game, go to **Options... -> Language**, and select **Medžuslovjansky**.

## 👥 How to Contribute

We highly appreciate community contributions to polish the vocabulary and grammar! 

* **Report Typos:** If you spot an incorrect translation, please open an [Issue](../../issues).
* **Suggest Improvements:** For better terminology matches, start a discussion in the Issues section.
* **Submit Fixes:** Fork the repository, apply your changes to the `.json` files, and submit a [Pull Request](../../pulls).

*Note: Please ensure all text files are saved using **UTF-8** encoding to preserve special characters.*
